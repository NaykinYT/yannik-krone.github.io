# Deploy to yannik-krone.github.io

1. Create a new GitHub repo named `yannik-krone.github.io` (no README)

2. Push from this folder:
   ```bash
   git remote add origin https://github.com/yannik-krone/yannik-krone.github.io.git
   git push -u origin main
   ```

3. GitHub Pages will serve the site at **https://yannik-krone.github.io**
