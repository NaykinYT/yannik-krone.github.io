Applied ML for Neural Information Encoding

Computational Neuroscience · Representation Learning

Research Assistant at EPFL's Laboratory of Physics of Biological Systems. Developing ML models to predict behavioral responses from neuronal activity.

Code not publicly available (academic/institutional constraints)
Click for details

I develop machine learning models at the Laboratory of Physics of Biological Systems (LPBS) under Prof. Sahand Jamal Rahi, to predict behavioral responses directly from neuronal activity. My focus is on the predictive modeling side: mapping time-series of neuronal activity to future and past behavior.

Working with C. elegans as a model organism, we use simultaneous neuronal and behavioral recordings from freely moving worms during chemotaxis. Second-layer interneurons (RIA, RIB, RIM) integrate sensory and internal signals; the goal is to understand how they encode behaviorally relevant information. My contribution centers on regression analyses: predicting the worm's current and future locomotion speed from interneuron activity. RIM and RIB together predict both forward and backward behavior; RIB alone emerges as a stronger predictor across individuals.

This sits at the intersection of applied machine learning, computational neuroscience, and time-series modeling—combining scalable learning systems with interpretable representations of how neural activity maps to behavior over time.

Separator-Aware SAT Encoding for Exact Treewidth

Algorithms · Graph Theory · SAT/MaxSAT

Developed a separator-aware encoding embedding safe graph decompositions into a single incremental SAT/MaxSAT framework. Supervised by Prof. Grossglauser and Prof. Thiran (Grade: 6/6).

Code Paper
Click for details

Treewidth is a central graph parameter measuring how "tree-like" a graph is; computing it exactly is NP-hard. This project implements a SAT-based exact treewidth solver that embeds safe graph decompositions into an incremental SAT/MaxSAT framework. The key innovation is a separator-aware encoding that uses safe separators (articulation points, clique separators, almost-cliques) to decompose the graph and integrate globally linked width constraints, preserving correctness across subinstances.

The implementation (Python 3.13+) supports multiple strategies: iterative binary search, iterative linear search, and MaxSAT—both in standard Berg–Järvisalo style and separator-aware variants. A hybrid solver combines separator decomposition with heuristic bounds and SAT enhancements. Benchmarked on PACE 2017 instances, Berg–Järvisalo graphs, and random graphs, the approach achieves consistent empirical speedups over baseline SAT encodings.

I fine-tuned Qwen3-0.6B into a domain-specialized STEM tutor, targeting efficient deployment for educational use cases. The pipeline combined Supervised Fine-Tuning (SFT), Direct Preference Optimization (DPO), quantization, and Retrieval-Augmented Generation (RAG) to balance model capability with resource constraints.

ML for Early Detection of Growth Faltering

Applied ML · Healthcare · Large-Scale Data

Modeled longitudinal malnutrition prediction using 16M+ medical consultations from Burkina Faso. Final model: 96.68% accuracy, 76.54% F1 on binary classification.

Paper Code & data not publicly available (academic/confidentiality constraints)
Click for details

This project addressed early detection of growth faltering (malnutrition) in children under five, using a large-scale longitudinal dataset of over 16 million medical consultations from Burkina Faso. The goal was to predict which children are at risk of growth faltering so interventions can be deployed earlier.

I evaluated multiple approaches—Logistic Regression, Random Forest, and LightGBM—for binary classification of growth faltering risk. The final model achieved 96.68% accuracy and 76.54% F1, balancing high recall for at-risk cases with precision. The work highlights the challenges of healthcare modeling at scale and the importance of interpretable, deployable models in resource-constrained settings. Code and data are not publicly available due to confidentiality constraints.

Secure Distributed Computing

Decentralized Systems Engineering · Distributed Systems · Cryptography · FHE

Implemented decentralized messaging, file sharing, and consensus in Go. Integrated privacy-preserving distributed learning with fully homomorphic encryption. Best among 29 projects of the whole class.

Report Code not publicly available (academic reasons)
Click for details

Built on the Peerster framework—a decentralized P2P system in Go supporting messaging, file sharing, and consensus—this project extended it with privacy-preserving federated learning. The challenge: train a global model across devices without any participant (including the aggregation server) seeing individual model updates.

Using xMK-CKKS (multi-key homomorphic encryption), each device encrypts its local model updates before sharing. The server aggregates ciphertexts and obtains only the averaged weights after collaborative decryption—never seeing any single device's contributions. Implementation includes message handlers for aggregation rounds, key management, failure detection via heartbeats, and asynchronous training. Tested under varying network topologies with 80.7% code coverage. Best project grade in class (1st among 29 groups) in a cohort of 100+ Master's students.

Proximity — Decentralized Trust Graph

SUI x BSA Hackathon 2024 · $20K Prize Pool

Designed a multi-component decentralized proximity detection system in 24h. Leveraged Bluetooth and blockchain for real-time, immutable on-chain trust graphs.

GitHub
Click for details

Proximity is a decentralized trust graph for presence verification, built in 24 hours for the SUI x BSA Hackathon 2024. The system uses Bluetooth for proximity detection and blockchain to record verifiable, immutable presence proofs—creating a real-time on-chain trust graph of who was physically near whom.

The architecture spans three components: a Java backend (proximity-server) for coordination and Bluetooth logic, a web demo (proximity-web-demo) for user interaction, and Move smart contracts (proximity-move) on the Sui blockchain for storing and verifying P2P presence proofs. Achieved 2nd place among 30+ international teams.

3rd Place

TrustLink — Secure Smart Contract Escrow

SUI x BSA Hackathon 2023 · $25K Prize Pool

Built a Web3 escrow system in 24h. Implemented smart contract logic using Move's object-ownership model for trustless execution.

Code
Click for details

TrustLink is a secure escrow system built in 24 hours for the Sui Blockchain Hackathon 2023. It enables trusted exchanges between multiple parties with specific roles, requirements, and conditional actions—all enforced on-chain.

The Move smart contracts define modular structures: TrustedContract (immutable rules), TrustedContractHandler (mutable state), ContractDescriptor (steps, requirements, actions, expiration). Participants put items into the contract, confirm steps, and items are released only when all requirements are met. Supports expiration handling, lock/unlock of assets, and a confirmation key mechanism to bridge on-chain logic with real-world verification. 3rd place in the international competition.
